======
ロボットビジョン2021-team7
======

腕立て伏せのフォームが悪いと本気でブチギレてくるプログラム
=====

USAGE
-----
事前にpip等で Poetry_ を入れること

``RobotVision2021-7`` ディレクトリで以下のコマンドを実行

環境構築
    ``poetry install``

実行
    ``poetry run python robotvision2021_7 -m``


VERIFY
-----
白背景に全身黒でトレーニングだ

.. _Poetry: https://github.com/python-poetry/poetry
